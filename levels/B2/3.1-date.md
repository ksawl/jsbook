## |1| **Создание даты**

Объект `Date` можно создать четырьмя основными способами:

1.  **Текущее время**: `new Date()`
2.  **Из временной метки (ms)**: `new Date(milliseconds)`
3.  **Из строки**: `new Date("2023-10-25")`
4.  **По компонентам**: `new Date(year, month, date, hours, minutes, seconds, ms)`

> [!IMPORTANT]
> В JavaScript месяцы считаются от **0** до **11**. Январь — 0, Декабрь — 11.

```javascript
const now = new Date();
const targetDate = new Date(2023, 9, 25); // 25 октября 2023
console.log(targetDate.toDateString()); // "Wed Oct 25 2023"
```

## |2| **Получение компонентов даты**

Методы получения данных всегда возвращают значения относительно локального часового пояса:

-   `getFullYear()` — год (4 цифры).
-   `getMonth()` — месяц (0-11).
-   `getDate()` — число месяца (1-31).
-   `getDay()` — день недели (0-6, где 0 — воскресенье).
-   `getHours()`, `getMinutes()`, `getSeconds()`, `getMilliseconds()`.

```javascript
const date = new Date();
console.log(`Год: ${date.getFullYear()}`);
console.log(`Месяц (0-indexed): ${date.getMonth()}`);
```

---

## |3| **Установка компонентов даты**

Для изменения даты используются методы `set*`:

-   `setFullYear(year, [month], [date])`
-   `setMonth(month, [date])`
-   `setDate(date)`

**Автоисправление дат:** JavaScript автоматически корректирует неправильные значения. Если установить "32 января", объект превратится в "1 февраля". Это удобно для вычисления дат (например, "через 70 дней от сегодня").

```javascript
const d = new Date(2023, 0, 1); // 1 янв 2023
d.setDate(32); 
console.log(d.toDateString()); // "Wed Feb 01 2023"
```

## |4| **Работа с временными метками (Timestamp)**

**Временная метка** — это количество миллисекунд, прошедших с 1 января 1970 года (Unix Epoch).

-   `Date.now()` — возвращает текущую метку (быстрее, чем `new Date().getTime()`).
-   `getTime()` — возвращает метку для конкретной даты.

Это основной способ измерения производительности или вычисления разницы между датами.

```javascript
const start = Date.now();
// ... какой-то тяжелый код ...
const end = Date.now();
console.log(`Выполнено за ${end - start} мс`);
```

## |5| **Разбор и форматирование**

### **Статический метод `Date.parse()`**
Позволяет получить timestamp из строки формата ISO 8601 (`YYYY-MM-DDTHH:mm:ss.sssZ`).

### **Современное форматирование: `Intl.DateTimeFormat`**
Для красивого отображения даты пользователю лучше использовать `Intl`, так как `toString()` выдает технический формат.

```javascript
const now = new Date();
const formatter = new Intl.DateTimeFormat('ru-RU', {
    day: 'numeric',
    month: 'long',
    year: 'numeric',
    weekday: 'short'
});

console.log(formatter.format(now)); // напр. "ср, 25 октября 2023 г."
```

---

## **Итог**

-   Объекты `Date` хранят время в миллисекундах с 1970 года.
-   Помните про **0-индексацию месяцев**.
-   Используйте `Date.now()` для замеров времени.
-   Полагайтесь на автоисправление дат для выполнения календарных расчетов.
-   Для локализации используйте `Intl.DateTimeFormat`.

## **Практика**

### 1. **Форматирование "ДД-ММ-ГГГГ"**
Напишите функцию `formatDate(date)`, которая принимает объект `Date` и возвращает строку в формате `ДД-ММ-ГГГГ` (например, "01-01-2023"). Используйте `padStart(2, '0')` для дополнения нулями.

### 2. **День недели через N дней**
Напишите функцию `getDayAfter(days)`, которая возвращает название дня недели (на русском), который наступит через `days` дней от текущего момента.

### 3. **Таймер выполнения**
Создайте скрипт, который замеряет время выполнения цикла от 0 до 1 000 000 и выводит результат в консоль.
