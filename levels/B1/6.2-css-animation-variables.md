## |1| **Плавные переходы (transition)**

Переходы позволяют плавно менять значение CSS-свойства (цвет, размер, прозрачность) при каком-то событии, например, при наведении курсора (`:hover`).

```css
.button {
  background-color: blue;
  transition: background-color 0.3s ease-in-out;
}

.button:hover {
  background-color: darkblue;
}
```

-   `ease-in-out` делает начало и конец анимации плавными.
-   Можно анимировать несколько свойств сразу: `transition: all 0.5s`.

---

## |2| **Сложные анимации (@keyframes)**

Если нужно, чтобы элемент двигался по сложной траектории или менялся бесконечно, используйте `@keyframes`.

```css
@keyframes slideIn {
  0% { transform: translateX(-100%); opacity: 0; }
  100% { transform: translateX(0); opacity: 1; }
}

.title {
  animation: slideIn 1s forwards;
}
```

-   **`infinity`** — позволяет делать бесконечные анимации (например, вращение).
-   **`forwards`** — оставляет элемент в конечном состоянии после завершения.

---

## |3| **CSS Переменные (Custom Properties)**

Переменные позволяют хранить цвета, шрифты и размеры в одном месте. Если вы решите изменить "фирменный цвет" своего сайта, вам придется менять его только в одной строке.

```css
:root {
  --main-color: #ff4500;
  --bg-color: #f4f4f4;
  --spacing: 15px;
}

.card {
  background-color: var(--bg-color);
  border: 1px solid var(--main-color);
  padding: var(--spacing);
}
```

---

## |4| **Оптимизация анимаций**

Не все свойства анимируются одинаково хорошо. Чтобы сайт не тормозил:
-   **Анимируйте** `transform` (перемещение, масштаб) и `opacity` (прозрачность). Они работают через видеокарту (GPU).
-   **Избегайте** анимации `width`, `height`, `margin` и `top/left/right/bottom`, так как они заставляют браузер пересчитывать всю страницу целиком (Reflow).

---

## **Итог**

-   **Transition** — для простых переходов "из точки А в точку Б".
-   **Keyframes** — для сложных циклов и многоэтапных движений.
-   **Переменные (--var)** — стандарт для создания тем и удобной поддержки кода.
-   Анимируйте **через transform**, чтобы сохранить плавность (60 FPS).

## **Практика**

### 1. **Кнопка с "пульсом"**
Создайте кнопку, которая бесконечно и плавно пульсирует (увеличивается и уменьшается в масштабе на 5-10%), используя `@keyframes` и `transform: scale()`.

### 2. **Темная и Светлая темы**
Создайте страницу с небольшим текстом. Используйте CSS-переменные для цвета текста и фона. Напишите одну кнопку на JS, которая при клике добавляет класс `dark` на тег `<body>`, переопределяя значения переменных.

### 3. **Вплывающий список**
Сделайте так, чтобы при загрузке страницы элементы списка `<li>` плавно появлялись снизу вверх один за другим с небольшой задержкой (`animation-delay`).

### 4. **Индикатор прогресса**
Создайте полосу загрузки. При наведении на её контейнер внутренняя полоса должна плавно заполняться на 100% за 2 секунды.
