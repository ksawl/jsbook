# **Глобальные объекты: Math и Date**

В JavaScript есть встроенные объекты, которые предоставляют готовые инструменты для математических вычислений и работы со временем. Эти объекты — **Math** и **Date**. Они доступны глобально и позволяют решать типовые задачи без написания сложных алгоритмов с нуля.

## |1| **Объект Math**

`Math` — это встроенный объект, содержащий математические константы и функции. У него нет конструктора (нельзя сделать `new Math()`), все его свойства и методы статические.

### 1. **Округление**
- `Math.floor(x)` — округление вниз (до ближайшего меньшего целого).
- `Math.ceil(x)` — округление вверх (до ближайшего большего целого).
- `Math.round(x)` — округление до ближайшего целого по правилам математики.
- `Math.trunc(x)` — удаление дробной части.

```javascript
console.log(Math.floor(3.9)); // 3
console.log(Math.ceil(3.1));  // 4
console.log(Math.round(3.5)); // 4
```

### 2. **Случайные числа**
- `Math.random()` — возвращает случайное число от 0 (включительно) до 1 (не включая 1).

**Формула для целого числа в диапазоне (min, max):**
```javascript
function getRandomInt(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}
```

### 3. **Другие полезные методы**
- `Math.max(a, b, c...)` / `Math.min(...)` — поиск максимума и минимума.
- `Math.pow(base, exp)` — возведение в степень (аналог `base ** exp`).
- `Math.sqrt(x)` — квадратный корень.
- `Math.abs(x)` — абсолютное значение (модуль).

## |2| **Объект Date**

Для работы с датами и временем используется объект `Date`. Он хранит время в миллисекундах, прошедших с 1 января 1970 года.

### 1. **Создание даты**
```javascript
const now = new Date(); // Текущая дата и время
const specificDate = new Date('2025-01-01'); // Из строки
const componentsDate = new Date(2025, 0, 1, 12, 0); // 1 янв 2025, 12:00
```
> [!IMPORTANT]
> Месяцы в JavaScript начинаются с **0** (январь — 0, декабрь — 11).

### 2. **Получение данных**
- `getFullYear()` — получить год (4 цифры).
- `getMonth()` — месяц (0-11).
- `getDate()` — день месяца (1-31).
- `getDay()` — день недели (0 - воскресенье, 6 - суббота).
- `getTime()` — timestamp в миллисекундах.

### 3. **Текущий Timestamp**
Метод `Date.now()` возвращает текущее время в миллисекундах. Это быстрее, чем создавать объект `new Date()`, и идеально подходит для замера времени выполнения кода.

```javascript
const start = Date.now();
// ... код ...
const end = Date.now();
console.log(`Затрачено: ${end - start} мс`);
```

## **Итог**

- **`Math`** предоставляет инструменты для работы с числами (округление, случайность, степени).
- **`Date`** используется для всего, что связано со временем.
- Помните, что **`Math`** — это просто объект-библиотека, а **`Date`** — это конструктор, требующий `new`.
- Будьте внимательны с **0-индексацией месяцев**.

## **Практика**

### 1. **Генератор случайных цветов**
Напишите функцию `getRandomColor()`, которая возвращает случайный цвет в формате RGB: `rgb(r, g, b)`, где каждое число от 0 до 255.

### 2. **Сколько дней до Нового Года?**
Напишите скрипт, который высчитывает количество полных дней, оставшихся до 1 января следующего года.

### 3. **Площадь круга**
Используя `Math.PI` и `Math.pow`, напишите функцию `getCircleArea(radius)`, которая возвращает площадь круга.
