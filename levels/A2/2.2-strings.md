## |1| **Шаблонные строки**

**Шаблонные строки** (Template Literals) позволяют создавать сложные строки с использованием обратных кавычек (`` ` ``).

### 1. **Интерполяция**
Вы можете вставлять переменные или выражения прямо в строку с помощью синтаксиса `${...}`.

```javascript
const name = 'Харизма';
const greeting = `Добро пожаловать, ${name}!`;
console.log(greeting); // Добро пожаловать, Харизма!
```

### 2. **Многострочность**
Обычные строки не могут занимать несколько строк без специальных символов. Шаблонные строки сохраняют переносы.

```javascript
const list = `
  1. Пункт
  2. Пункт
`;
```

## |2| **Базовые свойства и методы**

### 1. **Длина строки**
Свойство `.length` возвращает количество символов.

```javascript
console.log('JS'.length); // 2
```

### 2. **Изменение регистра**
- `.toUpperCase()` — все буквы в верхний регистр.
- `.toLowerCase()` — все буквы в нижний регистр.

### 3. **Удаление пробелов**
- `.trim()` — удаляет пробелы в начале и в конце строки.

```javascript
const input = '   login   ';
console.log(input.trim()); // 'login'
```

## |3| **Поиск подстроки**

### 1. **Проверка наличия**
- `.includes(substring)` — возвращает `true`, если подстрока найдена.
- `.startsWith(substring)` / `.endsWith(substring)` — проверяет начало или конец.

### 2. **Поиск индекса**
- `.indexOf(substring)` — возвращает индекс первого вхождения или `-1`.

```javascript
const text = 'Я люблю JavaScript';
console.log(text.includes('Java')); // true
console.log(text.indexOf('Java')); // 8
```

---

## |4| **Получение подстроки**

### 1. **Метод `slice(start, end)`**
Возвращает часть строки от `start` до `end` (не включая `end`).

### 2. **Метод `substring(start, end)`**
Похож на `slice`, но иначе обрабатывает отрицательные индексы.

```javascript
const str = 'Hello world';
console.log(str.slice(0, 5)); // 'Hello'
```

---

## |5| **Изменение и замена**

### 1. **Метод `replace(old, new)`**
Заменяет первое вхождение. Для замены всех используется `replaceAll` или регулярные выражения.

### 2. **Метод `split(separator)`**
Разбивает строку в массив по разделителю.

```javascript
const csv = 'Apple,Banana,Orange';
const fruits = csv.split(',');
console.log(fruits); // ['Apple', 'Banana', 'Orange']
```

---

## **Итог**

- Используйте **шаблонные строки** для удобной вставки переменных.
- Строки **неизменяемы**: методы не меняют исходную строку, а возвращают новую.
- **`split`** — превращает строку в массив, **`join`** (метод массива) — обратно.

## **Практика**

### 1. **Чистка ввода**
Напишите функцию `cleanInput(str)`, которая удаляет пробелы по краям и переводит всё в нижний регистр.

### 2. **Сокращение текста**
Напишите функцию `truncate(str, maxlength)`, которая проверяет длину строки. Если она превышает `maxlength`, заменяет конец строки на "...", чтобы общая длина была равна `maxlength`.

### 3. **Проверка Spam**
Напишите функцию `checkSpam(str)`, возвращающую `true`, если строка содержит 'buy' или 'free' (регистр не важен).
