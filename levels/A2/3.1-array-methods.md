## |1| **Метод `forEach`**

Самый простой метод перебора. Он просто выполняет функцию для каждого элемента массива.

```javascript
const fruits = ['Яблоко', 'Апельсин', 'Груша'];

fruits.forEach((item, index) => {
  console.log(`${index}: ${item}`);
});
```

**Особенности:**
- Возвращает `undefined`.
- Его нельзя прервать с помощью `break` или `continue`.

---

## |2| **Трансформация: `map`**

Метод `map` создает новый массив, состоящий из результатов вызова функции для каждого элемента.

```javascript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);

console.log(doubled); // [2, 4, 6, 8]
```

**Когда использовать:** Когда вам нужно превратить один список данных в другой (например, массив объектов в массив строк с именами).

---

## |3| **Фильтрация: `filter`**

Создает новый массив со всеми элементами, прошедшими проверку в функции.

```javascript
const ages = [12, 18, 25, 30, 15];
const adults = ages.filter(age => age >= 18);

console.log(adults); // [18, 25, 30]
```

---

## |4| **Свертка: `reduce`**

Самый гибкий и "умный" метод. Он используется для вычисления одного значения на основе всего массива.

```javascript
const prices = [100, 200, 300];
const total = prices.reduce((acc, current) => acc + current, 0);

console.log(total); // 600
```

- **`acc` (accumulator)**: Накапливает результат.
- **`0`**: Начальное значение аккумулятора.

---

## |5| **Поиск и проверка: `find`, `some`, `every`**

- **`.find(fn)`**: Возвращает **первый** подходящий элемент (не массив!).
- **`.some(fn)`**: Возвращает `true`, если хотя бы один элемент подходит.
- **`.every(fn)`**: Возвращает `true`, если **все** элементы подходят.

```javascript
const users = [{id: 1, name: 'Ivan'}, {id: 2, name: 'Anna'}];
const user = users.find(u => u.id === 2); // {id: 2, name: 'Anna'}
```

---

## **Итог**

- **Не меняйте исходные данные**: `map`, `filter` и `reduce` возвращают новый результат, не портя старый массив.
- Имена важны: используйте понятные названия для аргументов (например, `user => user.age` вместо `x => x.age`).
- Цепочки (Chaining): Методы можно соединять: `arr.filter(...).map(...)`.

## **Практика**

### 1. **Длины строк**
Есть массив `const strings = ["HTML", "CSS", "JavaScript"]`. С помощью `map` создайте массив, содержащий длины этих строк.

### 2. **Только положительные**
Отфильтруйте массив чисел `[-1, 2, 3, -5, 4]`, оставив только числа больше нуля.

### 3. **Общая стоимость**
Дан массив товаров:
```javascript
const cart = [
  { name: 'Книга', price: 500 },
  { name: 'Ручка', price: 50 },
  { name: 'Тетрадь', price: 100 }
];
```
Посчитайте общую сумму корзины с помощью `reduce`.
