# **2.4. Условия и циклы**

## Условные операторы

### Оператор `if`

Базовый условный оператор для выполнения кода при истинном условии:

```javascript
if (условие) {
  // код выполняется, если условие истинно
}
```

### Оператор `else`

Выполняется, когда условие в `if` ложно:

```javascript
if (условие) {
  // код для истинного условия
} else {
  // код для ложного условия
}
```

### Оператор `else if`

Позволяет проверить несколько условий последовательно:

```javascript
if (условие1) {
  // код для условие1
} else if (условие2) {
  // код для условие2
} else {
  // код, если все условия ложны
}
```

### Оператор `switch`

Альтернатива множественным `if-else` для сравнения значения с константами:

```javascript
switch (выражение) {
  case значение1:
    // код для значение1
    break;
  case значение2:
    // код для значение2
    break;
  default:
  // код по умолчанию
}
```

## Тернарный оператор

Сокращенная форма условного оператора:

```javascript
результат = условие ? значение_если_истина : значение_если_ложь;
```

## Циклы

### Цикл `for`

Используется, когда известно количество итераций:

```javascript
for (инициализация; условие; изменение) {
  // тело цикла
}
```

### Цикл `while`

Выполняется, пока условие истинно:

```javascript
while (условие) {
  // тело цикла
}
```

### Цикл `do...while`

Выполняется хотя бы один раз, затем проверяет условие:

```javascript
do {
  // тело цикла
} while (условие);
```

### Цикл `for...of`

Для перебора элементов массива или итерируемых объектов:

```javascript
for (const элемент of массив) {
  // работа с элементом
}
```

### Цикл `for...in`

Для перебора ключей объекта:

```javascript
for (const ключ in объект) {
  // работа с ключом
}
```

## Операторы управления циклом

### `break`

Прерывает выполнение цикла:

```javascript
for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i); // выведет 0, 1, 2, 3, 4
}
```

### `continue`

Пропускает текущую итерацию и переходит к следующей:

```javascript
for (let i = 0; i < 10; i++) {
  if (i % 2 === 0) continue;
  console.log(i); // выведет только нечетные числа
}
```

## Практические примеры

### Проверка четности числа

```javascript
const number = 42;

if (number % 2 === 0) {
  console.log("Число четное");
} else {
  console.log("Число нечетное");
}
```

### Определение времени суток

```javascript
const hour = new Date().getHours();

if (hour < 6) {
  console.log("Ночь");
} else if (hour < 12) {
  console.log("Утро");
} else if (hour < 18) {
  console.log("День");
} else {
  console.log("Вечер");
}
```

### Таблица умножения

```javascript
for (let i = 1; i <= 10; i++) {
  for (let j = 1; j <= 10; j++) {
    console.log(`${i} × ${j} = ${i * j}`);
  }
}
```

## Вложенные условия и циклы

Условия и циклы можно комбинировать и вкладывать друг в друга для создания сложной логики.
