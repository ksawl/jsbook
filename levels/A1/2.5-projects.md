# **2.5. Проекты для закрепления**

## Проект: "Hello, World!"

Создайте простой HTML-файл с JavaScript, который выводит приветствие в консоль.

### Задача

Написать скрипт на JavaScript, который выводит приветствие в консоль.

### Требования

1. Создать HTML-файл
2. Подключить JavaScript
3. Вывести в консоль сообщение "Hello, World!"

### Решение

**index.html:**

```html
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Hello World</title>
  </head>
  <body>
    <h1>Мой первый JavaScript</h1>
    <script src="script.js"></script>
  </body>
</html>
```

**script.js:**

```javascript
// Приветствие в консоли
console.log("Hello, World!");

// Дополнительные варианты
console.log("Привет, мир!");
console.log("Добро пожаловать в JavaScript!");

// Использование переменных
const greeting = "Hello, World!";
console.log(greeting);

// Использование функции
function sayHello() {
  console.log("Hello, World!");
}

sayHello();
```

## Дополнительные задания

### 1. Персональное приветствие

Создайте скрипт, который выводит персональное приветствие с вашим именем.

```javascript
const name = "Ваше имя";
console.log(`Привет, ${name}!`);
```

### 2. Математические операции

Создайте переменные с числами и выведите результаты различных операций:

```javascript
const a = 10;
const b = 5;

console.log(`Сложение: ${a} + ${b} = ${a + b}`);
console.log(`Вычитание: ${a} - ${b} = ${a - b}`);
console.log(`Умножение: ${a} × ${b} = ${a * b}`);
console.log(`Деление: ${a} ÷ ${b} = ${a / b}`);
```

### 3. Работа с датой

Выведите текущую дату и время:

```javascript
const now = new Date();
console.log(`Текущая дата: ${now.toLocaleDateString()}`);
console.log(`Текущее время: ${now.toLocaleTimeString()}`);
```

### 4. Условные операторы

Создайте скрипт, который проверяет время суток и выводит соответствующее приветствие:

```javascript
const hour = new Date().getHours();
let greeting;

if (hour < 6) {
  greeting = "Спокойной ночи!";
} else if (hour < 12) {
  greeting = "Доброе утро!";
} else if (hour < 18) {
  greeting = "Добрый день!";
} else {
  greeting = "Добрый вечер!";
}

console.log(greeting);
```

### 5. Цикл с выводом

Создайте цикл, который выводит числа от 1 до 10:

```javascript
for (let i = 1; i <= 10; i++) {
  console.log(`Число: ${i}`);
}
```
