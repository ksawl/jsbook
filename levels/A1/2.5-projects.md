## |1| **Проект: "Hello, World!"**

Создайте простой HTML-файл с JavaScript, который выводит приветствие в консоль. Это классический первый шаг в изучении любого языка.

### 1. **Задача**
Написать скрипт на JavaScript, который выводит приветствие в консоль и отображает его на странице.

### 2. **Требования**
1. Создать HTML-файл.
2. Подключить JavaScript одним из способов.
3. Вывести в консоль сообщение "Hello, World!".

### 3. **Решение**

**index.html:**
```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Hello World</title>
</head>
<body>
    <h1>Мой первый JavaScript</h1>
    <script src="script.js"></script>
</body>
</html>
```

**script.js:**
```javascript
console.log("Hello, World!");
```

## |2| **Проект: "Калькулятор возраста"**

Программа, которая принимает год рождения и вычисляет текущий возраст, проверяя совершеннолетие.

### 1. **Задача**
Используя переменные и условные операторы, определить возраст пользователя.

### 2. **Решение**

```javascript
const birthYear = 2005;
const currentYear = new Date().getFullYear();
const age = currentYear - birthYear;

console.log(`Ваш возраст: ${age}`);

if (age >= 18) {
    console.log("Вы совершеннолетний");
} else {
    console.log("Вы еще молоды!");
}
```

## |3| **Проект: "Конвертер валют"**

Простой конвертер, который переводит рубли в доллары по заданному курсу.

### 1. **Задача**
Создать переменные для суммы в рублях и курса доллара, вывести результат.

### 2. **Решение**

```javascript
const rub = 1000;
const rate = 92.5;
const usd = (rub / rate).toFixed(2);

console.log(`${rub} рублей = ${usd} долларов`);
```

## **Практика**

### 1. **Персональное приветствие**
Создайте скрипт, который выводит персональное приветствие с вашим именем. Используйте шаблонные строки.

### 2. **Математические операции**
Создайте две переменные `a` и `b`, присвойте им числа и выведите результаты всех арифметических операций в консоль.

### 3. **Проверка времени суток**
Напишите программу, которая выводит "Доброе утро", "Добрый день" или "Добрый вечер" в зависимости от значения переменной `hour`.
