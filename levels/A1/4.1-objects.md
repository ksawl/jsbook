# **4.1. Объекты**

## Что такое объекты?

Объект в JavaScript — это коллекция свойств, где каждое свойство определяется как пара "ключ-значение". Объекты позволяют группировать связанные данные и функциональность.

```javascript
// Простой объект
const person = {
  name: "Иван",
  age: 25,
  city: "Москва",
};
```

## Создание объектов

### 1. Литерал объекта

Самый распространенный способ создания объектов:

```javascript
const car = {
  brand: "Toyota",
  model: "Camry",
  year: 2023,
  color: "синий",
};
```

### 2. Конструктор Object

```javascript
const user = new Object();
user.name = "Мария";
user.email = "maria@example.com";
```

### 3. Функция-конструктор

```javascript
function Person(name, age) {
  this.name = name;
  this.age = age;
}

const person1 = new Person("Алексей", 30);
const person2 = new Person("Елена", 28);
```

## Доступ к свойствам объекта

### Точечная нотация

```javascript
const book = {
  title: "JavaScript для начинающих",
  author: "Иван Петров",
  pages: 300,
};

console.log(book.title); // "JavaScript для начинающих"
console.log(book.author); // "Иван Петров"
```

### Квадратные скобки

```javascript
// Полезно когда ключ содержит пробелы или вычисляется динамически
const student = {
  "first name": "Анна",
  "last name": "Смирнова",
  grade: 5,
};

console.log(student["first name"]); // "Анна"

// Динамический доступ
const property = "grade";
console.log(student[property]); // 5
```

## Изменение свойств объекта

### Добавление новых свойств

```javascript
const phone = {
  brand: "iPhone",
  model: "14",
};

phone.color = "черный"; // Добавление через точечную нотацию
phone["storage"] = "128GB"; // Добавление через квадратные скобки

console.log(phone);
// { brand: 'iPhone', model: '14', color: 'черный', storage: '128GB' }
```

### Изменение существующих свойств

```javascript
const product = {
  name: "Ноутбук",
  price: 50000,
};

product.price = 45000; // Изменение цены
console.log(product.price); // 45000
```

## Удаление свойств

### Оператор `delete`

```javascript
const laptop = {
  brand: "Dell",
  model: "XPS",
  year: 2023,
  color: "серебристый",
};

delete laptop.color;
console.log(laptop); // { brand: 'Dell', model: 'XPS', year: 2023 }
```

## Методы объектов

Объекты могут содержать функции как значения свойств. Такие функции называются методами.

```javascript
const calculator = {
  result: 0,

  add: function (number) {
    this.result += number;
    return this;
  },

  subtract: function (number) {
    this.result -= number;
    return this;
  },

  multiply: function (number) {
    this.result *= number;
    return this;
  },

  getValue: function () {
    return this.result;
  },

  reset: function () {
    this.result = 0;
    return this;
  },
};

// Использование методов
calculator.add(10).multiply(2).subtract(5);
console.log(calculator.getValue()); // 15
```

### Сокращенный синтаксис методов (ES6)

```javascript
const person = {
  name: "Дмитрий",
  age: 35,

  // Сокращенный синтаксис
  greet() {
    return `Привет, меня зовут ${this.name}`;
  },

  getAge() {
    return this.age;
  },

  celebrate() {
    this.age++;
    return `Теперь мне ${this.age} лет!`;
  },
};

console.log(person.greet()); // "Привет, меня зовут Дмитрий"
console.log(person.celebrate()); // "Теперь мне 36 лет!"
```

## Ключевое слово `this`

`this` ссылается на объект, в контексте которого вызывается метод.

```javascript
const user = {
  name: "Светлана",
  age: 32,

  introduce() {
    return `Меня зовут ${this.name}, мне ${this.age} года`;
  },

  updateAge(newAge) {
    this.age = newAge;
    return `Возраст обновлен на ${this.age}`;
  },
};

console.log(user.introduce()); // "Меня зовут Светлана, мне 32 года"
console.log(user.updateAge(33)); // "Возраст обновлен на 33"
```

## Проверка существования свойств

### Оператор `in`

```javascript
const animal = {
  type: "кот",
  name: "Мурзик",
  age: 3,
};

console.log("name" in animal); // true
console.log("breed" in animal); // false
```

### Метод `hasOwnProperty()`

```javascript
console.log(animal.hasOwnProperty("name")); // true
console.log(animal.hasOwnProperty("breed")); // false
```

### Проверка на `undefined`

```javascript
console.log(animal.name !== undefined); // true
console.log(animal.breed !== undefined); // false
```

## Перебор свойств объекта

### Цикл `for...in`

```javascript
const company = {
  name: "Tech Corp",
  employees: 150,
  founded: 2010,
  location: "Санкт-Петербург",
};

for (const key in company) {
  console.log(`${key}: ${company[key]}`);
}
// name: Tech Corp
// employees: 150
// founded: 2010
// location: Санкт-Петербург
```

### `Object.keys()`

```javascript
const keys = Object.keys(company);
console.log(keys); // ['name', 'employees', 'founded', 'location']

keys.forEach((key) => {
  console.log(`${key}: ${company[key]}`);
});
```

### `Object.values()`

```javascript
const values = Object.values(company);
console.log(values); // ['Tech Corp', 150, 2010, 'Санкт-Петербург']
```

### `Object.entries()`

```javascript
const entries = Object.entries(company);
console.log(entries);
// [['name', 'Tech Corp'], ['employees', 150], ['founded', 2010], ['location', 'Санкт-Петербург']]

entries.forEach(([key, value]) => {
  console.log(`${key}: ${value}`);
});
```

## Копирование объектов

### Поверхностное копирование

```javascript
const original = {
  name: "Оригинал",
  value: 100,
};

// Spread-оператор
const copy1 = { ...original };

// Object.assign()
const copy2 = Object.assign({}, original);

copy1.name = "Копия 1";
console.log(original.name); // "Оригинал" (не изменился)
console.log(copy1.name); // "Копия 1"
```

### Проблема с вложенными объектами

```javascript
const complexObject = {
  name: "Сложный объект",
  nested: {
    value: 42,
  },
};

const shallowCopy = { ...complexObject };
shallowCopy.nested.value = 99;

console.log(complexObject.nested.value); // 99 (изменился!)
```

## Практические примеры

### Система управления задачами

```javascript
const taskManager = {
  tasks: [],
  nextId: 1,

  addTask(description, priority = "medium") {
    const task = {
      id: this.nextId++,
      description: description,
      priority: priority,
      completed: false,
      createdAt: new Date(),
    };

    this.tasks.push(task);
    return task;
  },

  completeTask(id) {
    const task = this.tasks.find((t) => t.id === id);
    if (task) {
      task.completed = true;
      task.completedAt = new Date();
    }
    return task;
  },

  getTasks(filter = "all") {
    switch (filter) {
      case "completed":
        return this.tasks.filter((t) => t.completed);
      case "pending":
        return this.tasks.filter((t) => !t.completed);
      default:
        return this.tasks;
    }
  },

  getTasksByPriority(priority) {
    return this.tasks.filter((t) => t.priority === priority);
  },
};

// Использование
taskManager.addTask("Изучить объекты JavaScript", "high");
taskManager.addTask("Сделать домашнее задание", "medium");
taskManager.addTask("Прочитать книгу", "low");

console.log("Все задачи:", taskManager.getTasks());
console.log("Важные задачи:", taskManager.getTasksByPriority("high"));

taskManager.completeTask(1);
console.log("Завершенные задачи:", taskManager.getTasks("completed"));
```

### Библиотека книг

```javascript
const library = {
  books: [],

  addBook(title, author, isbn, year) {
    const book = {
      title,
      author,
      isbn,
      year,
      available: true,
      borrowedBy: null,
      borrowedDate: null,
    };

    this.books.push(book);
    return book;
  },

  findBook(query) {
    return this.books.filter(
      (book) =>
        book.title.toLowerCase().includes(query.toLowerCase()) ||
        book.author.toLowerCase().includes(query.toLowerCase())
    );
  },

  borrowBook(isbn, borrowerName) {
    const book = this.books.find((b) => b.isbn === isbn);

    if (!book) {
      return "Книга не найдена";
    }

    if (!book.available) {
      return "Книга уже выдана";
    }

    book.available = false;
    book.borrowedBy = borrowerName;
    book.borrowedDate = new Date();

    return `Книга "${book.title}" выдана пользователю ${borrowerName}`;
  },

  returnBook(isbn) {
    const book = this.books.find((b) => b.isbn === isbn);

    if (!book) {
      return "Книга не найдена";
    }

    if (book.available) {
      return "Книга не была выдана";
    }

    book.available = true;
    book.borrowedBy = null;
    book.borrowedDate = null;

    return `Книга "${book.title}" возвращена`;
  },
};

// Демонстрация использования
library.addBook("1984", "Джордж Оруэлл", "978-0-452-28423-4", 1949);
library.addBook("Война и мир", "Лев Толстой", "978-5-17-082568-5", 1869);

console.log(library.borrowBook("978-0-452-28423-4", "Иван Петров"));
console.log(library.findBook("Толстой"));
console.log(library.returnBook("978-0-452-28423-4"));
```
