–í JavaScript –º–Ω–æ–≥–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤. –ö–∞–∂–¥—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä –∏–º–µ–µ—Ç —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π –Ω–æ–º–µ—Ä –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–∞. –¢–æ—Ç, —É –∫–æ–≥–æ —ç—Ç–æ —á–∏—Å–ª–æ **–±–æ–ª—å—à–µ**, ‚Äì –≤—ã–ø–æ–ª–Ω–∏—Ç—Å—è **—Ä–∞–Ω—å—à–µ**. –ï—Å–ª–∏ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π, —Ç–æ –ø–æ—Ä—è–¥–æ–∫ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è ‚Äì **—Å–ª–µ–≤–∞ –Ω–∞–ø—Ä–∞–≤–æ**.

–û—Ç—Ä—ã–≤–æ–∫ –∏–∑ —Ç–∞–±–ª–∏—Ü—ã –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç–æ–≤ (**–Ω–µ—Ç –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –≤—Å—ë –∑–∞–ø–æ–º–∏–Ω–∞—Ç—å**, –æ–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç —É–Ω–∞—Ä–Ω—ã—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –≤—ã—à–µ, —á–µ–º —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏—Ö –±–∏–Ω–∞—Ä–Ω—ã—Ö):

## |3| **–õ–æ–≥–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã**

–í JavaScript –µ—Å—Ç—å —Å–µ–º—å –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤:

–ù–µ—Å–º–æ—Ç—Ä—è –Ω–∞ —Å–≤–æ—ë –Ω–∞–∑–≤–∞–Ω–∏–µ, –¥–∞–Ω–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –º–æ–≥—É—Ç –ø—Ä–∏–º–µ–Ω—è—Ç—å—Å—è –∫ –∑–Ω–∞—á–µ–Ω–∏—è–º –ª—é–±—ã—Ö —Ç–∏–ø–æ–≤. –ü–æ–ª—É—á–µ–Ω–Ω—ã–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã —Ç–∞–∫–∂–µ –º–æ–≥—É—Ç –∏–º–µ—Ç—å —Ä–∞–∑–ª–∏—á–Ω—ã–π —Ç–∏–ø.

### 1. **|| (–ò–õ–ò)**

–û–ø–µ—Ä–∞—Ç–æ—Ä ¬´–ò–õ–ò¬ª –≤—ã–≥–ª—è–¥–∏—Ç –∫–∞–∫ –¥–≤–æ–π–Ω–æ–π —Å–∏–º–≤–æ–ª –≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω–æ–π —á–µ—Ä—Ç—ã:


–¢—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω–æ –≤ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏–∏ –ò–õ–ò –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–æ —Ç–æ–ª—å–∫–æ –¥–ª—è –º–∞–Ω–∏–ø—É–ª–∏—Ä–æ–≤–∞–Ω–∏—è –±—É–ª–µ–≤—ã–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏: –≤ —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ –∫–∞–∫–æ–π-–ª–∏–±–æ –∏–∑ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤ `true`, –æ–Ω –≤–µ—Ä–Ω—ë—Ç `true`, –≤ –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ–π —Å–∏—Ç—É–∞—Ü–∏–∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è `false`.

–í JavaScript, –∫–∞–∫ –º—ã —É–≤–∏–¥–∏–º –¥–∞–ª–µ–µ, —ç—Ç–æ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä —Ä–∞–±–æ—Ç–∞–µ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ –∏–Ω—ã–º –æ–±—Ä–∞–∑–æ–º. –ù–æ –¥–∞–≤–∞–π—Ç–µ —Å–ø–µ—Ä–≤–∞ –ø–æ—Å–º–æ—Ç—Ä–∏–º, —á—Ç–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Å –±—É–ª–µ–≤—ã–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏.

–°—É—â–µ—Å—Ç–≤—É–µ—Ç –≤—Å–µ–≥–æ —á–µ—Ç—ã—Ä–µ –≤–æ–∑–º–æ–∂–Ω—ã–µ –ª–æ–≥–∏—á–µ—Å–∫–∏–µ –∫–æ–º–±–∏–Ω–∞—Ü–∏–∏:

–ö–∞–∫ –º—ã –º–æ–∂–µ–º –Ω–∞–±–ª—é–¥–∞—Ç—å, —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–ø–µ—Ä–∞—Ü–∏–π –≤—Å–µ–≥–¥–∞ —Ä–∞–≤–µ–Ω `true`, –∑–∞ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ–º —Å–ª—É—á–∞—è, –∫–æ–≥–¥–∞ –æ–±–∞ –∞—Ä–≥—É–º–µ–Ω—Ç–∞ `false`.

–ï—Å–ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–µ –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ —Ç–∏–ø–∞, —Ç–æ –æ–Ω–æ –∫ –Ω–µ–º—É –ø—Ä–∏–≤–æ–¥–∏—Ç—Å—è –≤ —Ü–µ–ª—è—Ö –≤—ã—á–∏—Å–ª–µ–Ω–∏–π.

–ù–∞–ø—Ä–∏–º–µ—Ä, —á–∏—Å–ª–æ `1` –±—É–¥–µ—Ç –≤–æ—Å–ø—Ä–∏–Ω—è—Ç–æ –∫–∞–∫ `true`, –∞ `0` ‚Äì –∫–∞–∫ `false`:

### 2. **–ò–õ–ò `"||"` –Ω–∞—Ö–æ–¥–∏—Ç –ø–µ—Ä–≤–æ–µ –∏—Å—Ç–∏–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ**

–≠—Ç–æ –¥–µ–ª–∞–µ—Ç –≤–æ–∑–º–æ–∂–Ω—ã–º –±–æ–ª–µ–µ –∏–Ω—Ç–µ—Ä–µ—Å–Ω–æ–µ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ø–æ —Å—Ä–∞–≤–Ω–µ–Ω–∏—é —Å ¬´—á–∏—Å—Ç—ã–º, —Ç—Ä–∞–¥–∏—Ü–∏–æ–Ω–Ω—ã–º, —Ç–æ–ª—å–∫–æ –±—É–ª–µ–≤—ã–º –ò–õ–ò¬ª.

### 3. **||= (–õ–æ–≥–∏—á–µ—Å–∫–æ–µ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ –ò–õ–ò)**

> [!CAUTION]
>
> **–ù–æ–≤–∞—è –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å**
> –≠—Ç–∞ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –±—ã–ª–∞ –¥–æ–±–∞–≤–ª–µ–Ω–∞ –≤ —è–∑—ã–∫ –Ω–µ–¥–∞–≤–Ω–æ. –í —Å—Ç–∞—Ä—ã—Ö –±—Ä–∞—É–∑–µ—Ä–∞—Ö –º–æ–∂–µ—Ç –ø–æ–Ω–∞–¥–æ–±–∏—Ç—å—Å—è –ø–æ–ª–∏—Ñ–∏–ª.

–û–ø–µ—Ä–∞—Ç–æ—Ä –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è –ò–õ–ò `||=` –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –∫–∞–∫ –æ–±—ã—á–Ω—ã–π –ò–õ–ò `||` —Å –¥–æ–±–∞–≤–ª–µ–Ω–∏–µ–º —Å–∏–º–≤–æ–ª–∞ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è `=`. –¢–∞–∫–∞—è –∑–∞–ø–∏—Å—å –Ω–µ —Å–ª—É—á–∞–π–Ω–∞, —Ç–∞–∫ –∫–∞–∫ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –¥–∞–Ω–Ω–æ–≥–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –Ω–∞–ø—Ä—è–º—É—é –∑–∞–≤–∏—Å–∏—Ç –æ—Ç –¥–µ–π—Å—Ç–≤–∏–π —É–∂–µ –∏–∑–≤–µ—Å—Ç–Ω–æ–≥–æ –Ω–∞–º `||`.

–í–æ—Ç –µ–≥–æ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å:


–û–ø–µ—Ä–∞—Ç–æ—Ä `||=` –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –¥–≤–∞ –æ–ø–µ—Ä–∞–Ω–¥–∞ –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Å–ª–µ–¥—É—é—â–∏–µ –¥–µ–π—Å—Ç–≤–∏—è:

- –í—ã—á–∏—Å–ª—è–µ—Ç –æ–ø–µ—Ä–∞–Ω–¥—ã —Å–ª–µ–≤–∞ –Ω–∞–ø—Ä–∞–≤–æ.

–ö–æ–Ω—Ü–µ–ø—Ü–∏—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ `||=` –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ ¬´—Å–æ–∫—Ä–∞—â—ë–Ω–Ω–æ–º –≤—ã—á–∏—Å–ª–µ–Ω–∏–∏¬ª, –ø—Ä–∏–Ω—Ü–∏–ø —Ä–∞–±–æ—Ç—ã –∫–æ—Ç–æ—Ä–æ–≥–æ –º—ã —Ä–∞–∑–æ–±—Ä–∞–ª–∏ —Ä–∞–Ω–µ–µ.

–¢–µ–ø–µ—Ä—å –¥–∞–≤–∞–π—Ç–µ –ø–µ—Ä–µ–ø–∏—à–µ–º `a ||= b` –ø–æ–¥ –≤–∏–¥ ¬´—Å–æ–∫—Ä–∞—â—ë–Ω–Ω–æ–≥–æ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è¬ª:

### 4. **&& (–ò)**

–û–ø–µ—Ä–∞—Ç–æ—Ä –ò –ø–∏—à–µ—Ç—Å—è –∫–∞–∫ –¥–≤–∞ –∞–º–ø–µ—Ä—Å–∞–Ω–¥–∞ `&&`:

–ö–∞–∫ –∏ –≤ —Å–ª—É—á–∞–µ —Å –ò–õ–ò, –ª—é–±–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–æ–ø—É—Å–∫–∞–µ—Ç—Å—è –≤ –∫–∞—á–µ—Å—Ç–≤–µ –æ–ø–µ—Ä–∞–Ω–¥–∞ –ò:

```js
  // –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –∫–∞–∫ true && false
```

### 5. **–ò `¬´&&¬ª` –Ω–∞—Ö–æ–¥–∏—Ç –ø–µ—Ä–≤–æ–µ –ª–æ–∂–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ**

```js
alert(0 && "–∫–∞–∫–∞—è-—Ç–æ —Å—Ç—Ä–æ–∫–∞"); // 0
```

–ú–æ–∂–Ω–æ –ø–µ—Ä–µ–¥–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –∑–Ω–∞—á–µ–Ω–∏–π –ø–æ–¥—Ä—è–¥. –í —Ç–∞–∫–æ–º —Å–ª—É—á–∞–µ –≤–æ–∑–≤—Ä–∞—Ç–∏—Ç—Å—è –ø–µ—Ä–≤–æ–µ ¬´–ª–æ–∂–Ω–æ–µ¬ª –∑–Ω–∞—á–µ–Ω–∏–µ, –Ω–∞ –∫–æ—Ç–æ—Ä–æ–º –æ—Å—Ç–∞–Ω–æ–≤–∏–ª–∏—Å—å –≤—ã—á–∏—Å–ª–µ–Ω–∏—è.

```js

```

–ö–æ–≥–¥–∞ –≤—Å–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤–µ—Ä–Ω—ã, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è –ø–æ—Å–ª–µ–¥–Ω–µ–µ

```js

```

> [!NOTE]
>
> **–ü—Ä–∏–æ—Ä–∏—Ç–µ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ `&&` –±–æ–ª—å—à–µ, —á–µ–º —É `||`**
> –ü—Ä–∏–æ—Ä–∏—Ç–µ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ò `&&` –±–æ–ª—å—à–µ, —á–µ–º –ò–õ–ò `||`, —Ç–∞–∫ —á—Ç–æ –æ–Ω –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Ä–∞–Ω—å—à–µ.
>
> –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –∫–æ–¥ a `&& b || c && d` –ø–æ —Å—É—â–µ—Å—Ç–≤—É —Ç–∞–∫–æ–π –∂–µ, –∫–∞–∫ –µ—Å–ª–∏ –±—ã –≤—ã—Ä–∞–∂–µ–Ω–∏—è `&&` –±—ã–ª–∏ –≤ –∫—Ä—É–≥–ª—ã—Ö —Å–∫–æ–±–∫–∞—Ö: `(a && b) || (c && d)`.

–±—É–¥—å—Ç–µ –≤–Ω–∏–º–∞—Ç–µ–ª—å–Ω—ã

> [!WARNING]
>
> **–ù–µ –∑–∞–º–µ–Ω—è–π—Ç–µ `if` –Ω–∞ `||` –∏–ª–∏ `&&`**
> –ò–Ω–æ–≥–¥–∞ –ª—é–¥–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä –ò `&&` –∫–∞–∫ ¬´–±–æ–ª–µ–µ –∫–æ—Ä–æ—Ç–∫–∏–π —Å–ø–æ—Å–æ–± –∑–∞–ø–∏—Å–∏ `if`-–≤—ã—Ä–∞–∂–µ–Ω–∏—è¬ª.
>
> –ù–∞–ø—Ä–∏–º–µ—Ä:
>
> ```js
> let x = 1;
>
> x > 0 && alert("x –±–æ–ª—å—à–µ –Ω—É–ª—è!");
> ```
>
> –ò–Ω—Å—Ç—Ä—É–∫—Ü–∏—è –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ `&&` –±—É–¥–µ—Ç –≤—ã–ø–æ–ª–Ω–µ–Ω–∞ —Ç–æ–ª—å–∫–æ –≤ —Ç–æ–º —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ –¥–æ–π–¥–µ—Ç –¥–æ –Ω–µ–µ. –¢–æ –µ—Å—Ç—å, —Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ `(x > 0)` –∏—Å—Ç–∏–Ω–Ω–æ.
>
> –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –º—ã –∏–º–µ–µ–º –∞–Ω–∞–ª–æ–≥ –¥–ª—è —Å–ª–µ–¥—É—é—â–µ–≥–æ –∫–æ–¥–∞:
>
> ```js
> let x = 1;
>
> if (x > 0) alert("x –±–æ–ª—å—à–µ –Ω—É–ª—è!");
> ```
>
> –ù–µ—Å–º–æ—Ç—Ä—è –Ω–∞ —Ç–æ, —á—Ç–æ –≤–∞—Ä–∏–∞–Ω—Ç —Å `&&` –∫–∞–∂–µ—Ç—Å—è –±–æ–ª–µ–µ –∫–æ—Ä–æ—Ç–∫–∏–º, `if` –±–æ–ª–µ–µ –Ω–∞–≥–ª—è–¥–µ–Ω –∏, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, –±–æ–ª–µ–µ —á–∏—Ç–∞–±–µ–ª–µ–Ω. –ü–æ—ç—Ç–æ–º—É –º—ã —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–∂–¥—É—é –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—é –ø–æ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏—é: –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `if`, –µ—Å–ª–∏ –Ω–∞–º –Ω—É–∂–Ω–æ `if`, –∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `&&`, –µ—Å–ª–∏ –Ω–∞–º –Ω—É–∂–Ω–æ –ò.

### 6. **&&= (–õ–æ–≥–∏—á–µ—Å–∫–æ–µ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ –ò)**

> [!WARNING]
>
> –ù–æ–≤–∞—è –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å
> –≠—Ç–∞ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –±—ã–ª–∞ –¥–æ–±–∞–≤–ª–µ–Ω–∞ –≤ —è–∑—ã–∫ –Ω–µ–¥–∞–≤–Ω–æ. –í —Å—Ç–∞—Ä—ã—Ö –±—Ä–∞—É–∑–µ—Ä–∞—Ö –º–æ–∂–µ—Ç –ø–æ–Ω–∞–¥–æ–±–∏—Ç—å—Å—è –ø–æ–ª–∏—Ñ–∏–ª.

–û–ø–µ—Ä–∞—Ç–æ—Ä –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è –ò `&&=` –∑–∞–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –∫–∞–∫ –¥–≤–∞ –∞–º–ø–µ—Ä—Å–∞–Ω–¥–∞ `&&` –∏ —Å–∏–º–≤–æ–ª –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è `=`.

–í–æ—Ç –µ–≥–æ —Å–∏–Ω—Ç–∞–∫—Å–∏—Å:

```js
a &&= b;
```

–ü—Ä–∏–Ω—Ü–∏–ø –¥–µ–π—Å—Ç–≤–∏—è `&&=` –ø—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∏ —Ç–∞–∫–æ–π –∂–µ, –∫–∞–∫ –∏ —É –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è –ò–õ–ò `||=`. –ï–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–æ–µ –æ—Ç–ª–∏—á–∏–µ –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ `&&=` –ø—Ä–∏—Å–≤–æ–∏—Ç `a` –∑–Ω–∞—á–µ–Ω–∏–µ `b` —Ç–æ–ª—å–∫–æ –≤ —Ç–æ–º —Å–ª—É—á–∞–µ, –µ—Å–ª–∏ `a` –∏—Å—Ç–∏–Ω–Ω–æ.

–ö–æ–Ω—Ü–µ–ø—Ü–∏—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è –ò `&&=` —Ç–∞–∫–∂–µ –æ—Å–Ω–æ–≤—ã–≤–∞–µ—Ç—Å—è –Ω–∞ ¬´—Å–æ–∫—Ä–∞—â—ë–Ω–Ω–æ–º –≤—ã—á–∏—Å–ª–µ–Ω–∏–∏¬ª:

```js

```

–ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è:

```js
let greeting = "–ü—Ä–∏–≤–µ—Ç"; // —Å—Ç—Ä–æ–∫–∞ –Ω–µ–ø—É—Å—Ç–∞—è, –ø–æ—ç—Ç–æ–º—É –±—É–¥–µ—Ç –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∞ –∫ –ª–æ–≥–∏—á–µ—Å–∫–æ–º—É –∑–Ω–∞—á–µ–Ω–∏—é true –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º &&=
```

–¢–∞–∫ –∫–∞–∫ –æ–ø–µ—Ä–∞—Ç–æ—Ä –ª–æ–≥–∏—á–µ—Å–∫–æ–≥–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è –ò `&&=` —Ç–∞–∫–∂–µ –∫–∞–∫ –∏ `||=` —è–≤–ª—è–µ—Ç—Å—è ¬´—Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–∏–º —Å–∞—Ö–∞—Ä–æ–º¬ª, –º—ã –º–æ–∂–µ–º –±–µ–∑ –ø—Ä–æ–±–ª–µ–º –ø–µ—Ä–µ–ø–∏—Å–∞—Ç—å –ø—Ä–∏–º–µ—Ä –≤—ã—à–µ —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –ø—Ä–∏–≤—ã—á–Ω–æ–≥–æ –¥–ª—è –Ω–∞—Å `if`:

```js

  greeting = greeting + ", –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å!";
```

–ù–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ, –≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç `||=`, –æ–ø–µ—Ä–∞—Ç–æ—Ä `&&=` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ —Ä–µ–¥–∫–æ ‚Äì –æ–±—ã—á–Ω–æ, –≤ –∫–æ–º–±–∏–Ω–∞—Ü–∏–∏ —Å –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã–º–∏ —è–∑—ã–∫–æ–≤—ã–º–∏ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏—è–º–∏. –ü–æ–¥–æ–±—Ä–∞—Ç—å –∫–æ–Ω—Ç–µ–∫—Å—Ç –¥–ª—è –ø—Ä–∏–º–µ–Ω–µ–Ω–∏—è –¥–∞–Ω–Ω–æ–≥–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ ‚Äì –¥–æ–≤–æ–ª—å–Ω–æ –Ω–µ–ø—Ä–æ—Å—Ç–∞—è –∑–∞–¥–∞—á–∞.

### 7. **! (–ù–ï)**

–û–ø–µ—Ä–∞—Ç–æ—Ä –ù–ï –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω –≤–æ—Å–∫–ª–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–º –∑–Ω–∞–∫–æ–º `!`.

–°–∏–Ω—Ç–∞–∫—Å–∏—Å –¥–æ–≤–æ–ª—å–Ω–æ –ø—Ä–æ—Å—Ç:

```js

```

–û–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –æ–¥–∏–Ω –∞—Ä–≥—É–º–µ–Ω—Ç –∏ –≤—ã–ø–æ–ª–Ω—è–µ—Ç —Å–ª–µ–¥—É—é—â–∏–µ –¥–µ–π—Å—Ç–≤–∏—è:

- –°–Ω–∞—á–∞–ª–∞ –ø—Ä–∏–≤–æ–¥–∏—Ç –∞—Ä–≥—É–º–µ–Ω—Ç –∫ –ª–æ–≥–∏—á–µ—Å–∫–æ–º—É —Ç–∏–ø—É `true/false`.
- –ó–∞—Ç–µ–º –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø—Ä–æ—Ç–∏–≤–æ–ø–æ–ª–æ–∂–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.

–í —á–∞—Å—Ç–Ω–æ—Å—Ç–∏, –¥–≤–æ–π–Ω–æ–µ –ù–ï `!!` –∏—Å–ø–æ–ª—å–∑—É—é—Ç –¥–ª—è –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–π –∫ –ª–æ–≥–∏—á–µ—Å–∫–æ–º—É —Ç–∏–ø—É:

–¢–æ –µ—Å—Ç—å –ø–µ—Ä–≤–æ–µ –ù–ï –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –ª–æ–≥–∏—á–µ—Å–∫–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –æ–±—Ä–∞—Ç–Ω–æ–µ, –∞ –≤—Ç–æ—Ä–æ–µ –ù–ï —Å–Ω–æ–≤–∞ –∏–Ω–≤–µ—Ä—Ç–∏—Ä—É–µ—Ç –µ–≥–æ. –í –∫–æ–Ω—Ü–µ –º—ã –∏–º–µ–µ–º –ø—Ä–æ—Å—Ç–æ–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤ –ª–æ–≥–∏—á–µ—Å–∫–æ–µ.

–ï—Å—Ç—å –Ω–µ–º–Ω–æ–≥–æ –±–æ–ª–µ–µ –ø–æ–¥—Ä–æ–±–Ω—ã–π —Å–ø–æ—Å–æ–± —Å–¥–µ–ª–∞—Ç—å —Ç–æ –∂–µ —Å–∞–º–æ–µ ‚Äì –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è `Boolean`:

–ü—Ä–∏–æ—Ä–∏—Ç–µ—Ç –ù–ï `!` —è–≤–ª—è–µ—Ç—Å—è –Ω–∞–∏–≤—ã—Å—à–∏–º –∏–∑ –≤—Å–µ—Ö –ª–æ–≥–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤, –ø–æ—ç—Ç–æ–º—É –æ–Ω –≤—Å–µ–≥–¥–∞ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –ø–µ—Ä–≤—ã–º, –ø–µ—Ä–µ–¥ `&&` –∏–ª–∏ `||`.

## |4| **–û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è**

–û–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è (`=`) –≤ JavaScript –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è –∑–Ω–∞—á–µ–Ω–∏–π –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º. –û–¥–Ω–∞–∫–æ, –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ç–∏–ø–∞ –¥–∞–Ω–Ω—ã—Ö (–ø—Ä–∏–º–∏—Ç–∏–≤—ã –∏–ª–∏ –æ–±—ä–µ–∫—Ç—ã), –ø–æ–≤–µ–¥–µ–Ω–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è –º–æ–∂–µ—Ç —Ä–∞–∑–ª–∏—á–∞—Ç—å—Å—è: –≤ –æ–¥–Ω–∏—Ö —Å–ª—É—á–∞—è—Ö –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç **–∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è**, –∞ –≤ –¥—Ä—É–≥–∏—Ö ‚Äî **–∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ —Å—Å—ã–ª–∫–∏**. –î–∞–≤–∞–π—Ç–µ —Ä–∞–∑–±–µ—Ä–µ–º —ç—Ç–æ –ø–æ–¥—Ä–æ–±–Ω–æ.

### 1. **–ü—Ä–∏–º–∏—Ç–∏–≤–Ω—ã–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö (–∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è)**

–ö–æ–≥–¥–∞ –≤—ã –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç–µ –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é, —Å–æ–¥–µ—Ä–∂–∞—â—É—é –ø—Ä–∏–º–∏—Ç–∏–≤–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –¥—Ä—É–≥–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π, –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç **–∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è**.

–ü–æ–¥–æ–±–Ω—ã–µ –∫—Ä–∞—Ç–∫–∏–µ —Ñ–æ—Ä–º—ã –∑–∞–ø–∏—Å–∏ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –¥–ª—è –≤—Å–µ—Ö –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –∏ –ø–æ–±–∏—Ç–æ–≤—ã—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤: `/=`, `-=`, `**=` –∏ —Ç–∞–∫ –¥–∞–ª–µ–µ.

–í—ã–∑–æ–≤ —Å –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ–º –∏–º–µ–µ—Ç –≤ —Ç–æ—á–Ω–æ—Å—Ç–∏ —Ç–∞–∫–æ–π –∂–µ –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç, –∫–∞–∫ –æ–±—ã—á–Ω–æ–µ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ, —Ç–æ –µ—Å—Ç—å –≤—ã–ø–æ–ª–Ω–∏—Ç—Å—è –ø–æ—Å–ª–µ –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–∞ –¥—Ä—É–≥–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π:

### 8. **–ò–Ω–∫—Ä–µ–º–µ–Ω—Ç/–¥–µ–∫—Ä–µ–º–µ–Ω—Ç**

–û–¥–Ω–æ–π –∏–∑ –Ω–∞–∏–±–æ–ª–µ–µ —á–∞—Å—Ç—ã—Ö —á–∏—Å–ª–æ–≤—ã—Ö –æ–ø–µ—Ä–∞—Ü–∏–π —è–≤–ª—è–µ—Ç—Å—è —É–≤–µ–ª–∏—á–µ–Ω–∏–µ –∏–ª–∏ —É–º–µ–Ω—å—à–µ–Ω–∏–µ –Ω–∞ –µ–¥–∏–Ω–∏—Ü—É.

–î–ª—è —ç—Ç–æ–≥–æ —Å—É—â–µ—Å—Ç–≤—É—é—Ç –¥–∞–∂–µ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã:

#### **–ò–Ω–∫—Ä–µ–º–µ–Ω—Ç** `++` —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é –Ω–∞ `1`


> [!IMPORTANT]
>
> **–í–∞–∂–Ω–æ:**
> –ò–Ω–∫—Ä–µ–º–µ–Ω—Ç/–¥–µ–∫—Ä–µ–º–µ–Ω—Ç –º–æ–∂–Ω–æ –ø—Ä–∏–º–µ–Ω–∏—Ç—å —Ç–æ–ª—å–∫–æ –∫ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π. –ü–æ–ø—ã—Ç–∫–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –µ–≥–æ –Ω–∞ –∑–Ω–∞—á–µ–Ω–∏–∏, —Ç–∏–ø–∞ `5++`, –ø—Ä–∏–≤–µ–¥—ë—Ç –∫ **–æ—à–∏–±–∫–µ**.

#### **–ü—Ä–µ—Ñ–∏–∫—Å / –ü–æ—Å—Ç—Ñ–∏–∫—Å**

–û–ø–µ—Ä–∞—Ç–æ—Ä—ã `++` –∏ `--` –º–æ–≥—É—Ç –±—ã—Ç—å —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω—ã –Ω–µ —Ç–æ–ª—å–∫–æ –ø–æ—Å–ª–µ, –Ω–æ –∏ –¥–æ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π.

–ö–æ–≥–¥–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä –∏–¥—ë—Ç –ø–æ—Å–ª–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π ‚Äî —ç—Ç–æ **¬´–ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞¬ª**: `counter++`.
**¬´–ü—Ä–µ—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞¬ª** ‚Äî —ç—Ç–æ –∫–æ–≥–¥–∞ –æ–ø–µ—Ä–∞—Ç–æ—Ä –∏–¥—ë—Ç –ø–µ—Ä–µ–¥ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π: `++counter`.
–û–±–µ —ç—Ç–∏ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ –¥–µ–ª–∞—é—Ç –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ: —É–≤–µ–ª–∏—á–∏–≤–∞—é—Ç `counter` –Ω–∞ `1`.

–ï—Å—Ç—å –ª–∏ —Ä–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É –Ω–∏–º–∏? –î–∞, –Ω–æ —É–≤–∏–¥–µ—Ç—å –µ—ë –º—ã —Å–º–æ–∂–µ–º, —Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ –±—É–¥–µ–º –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç `++/--`.

–î–∞–≤–∞–π—Ç–µ –ø—Ä–æ—è—Å–Ω–∏–º —ç—Ç–æ—Ç –º–æ–º–µ–Ω—Ç. –ö–∞–∫ –º—ã –∑–Ω–∞–µ–º, –≤—Å–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –∑–Ω–∞—á–µ–Ω–∏–µ. –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –∏–Ω–∫—Ä–µ–º–µ–Ω—Ç–∞/–¥–µ–∫—Ä–µ–º–µ–Ω—Ç–∞ –Ω–µ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ. –ü—Ä–µ—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –≤ —Ç–æ –≤—Ä–µ–º—è –∫–∞–∫ –ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Ç–∞—Ä–æ–µ (–¥–æ —É–≤–µ–ª–∏—á–µ–Ω–∏—è/—É–º–µ–Ω—å—à–µ–Ω–∏—è —á–∏—Å–ª–∞).

–ß—Ç–æ–±—ã —É–≤–∏–¥–µ—Ç—å —Ä–∞–∑–Ω–∏—Ü—É, –≤–æ—Ç –Ω–µ–±–æ–ª—å—à–æ–π –ø—Ä–∏–º–µ—Ä:

```js

```

–í —Å—Ç—Ä–æ–∫–µ `(*)` –ø—Ä–µ—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞ `++counter` —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç `counter` –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –Ω–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ `2`. –¢–∞–∫ —á—Ç–æ `alert` –ø–æ–∫–∞–∂–µ—Ç `2`.

–¢–µ–ø–µ—Ä—å –ø–æ—Å–º–æ—Ç—Ä–∏–º –Ω–∞ –ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω—É—é —Ñ–æ—Ä–º—É:

```js

```

–í —Å—Ç—Ä–æ–∫–µ `(*)` –ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω–∞—è —Ñ–æ—Ä–º–∞ `counter++` —Ç–∞–∫–∂–µ —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç `counter`, –Ω–æ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å—Ç–∞—Ä–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ (–∫–æ—Ç–æ—Ä–æ–µ –±—ã–ª–æ –¥–æ —É–≤–µ–ª–∏—á–µ–Ω–∏—è). –¢–∞–∫ —á—Ç–æ `alert` –ø–æ–∫–∞–∂–µ—Ç `1`.

–ß–∞—Å—Ç–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä –∏–Ω–∫—Ä–µ–º–µ–Ω—Ç–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ —Ü–∏–∫–ª–µ `for`


–∑–¥–µ—Å—å –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è `i` —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç—Å—è –Ω–∞ `1` –ø–æ—Å–ª–µ –ø—Ä–æ—Ö–æ–¥–∂–µ–Ω–∏—è —Ü–∏–∫–ª–∞, –µ—Å–ª–∏ –Ω–∞–ø–∏—Å–∞—Ç—å –≤–º–µ—Å—Ç–æ `i++` `++i`, —Ç–æ –æ–Ω–∞ –±—É–¥–µ—Ç —É–≤–µ–ª–∏—á–∏–≤–∞—Ç—å—Å—è –ø–µ—Ä–µ–¥ –Ω–∞—á–∞–ª–æ–º —Ü–∏–∫–ª–∞

#### **–ü–æ–¥–≤–µ–¥—ë–º –∏—Ç–æ–≥–∏**

–ï—Å–ª–∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –Ω–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è, –∞ –Ω—É–∂–Ω–æ —Ç–æ–ª—å–∫–æ —É–≤–µ–ª–∏—á–∏—Ç—å/—É–º–µ–Ω—å—à–∏—Ç—å –ø–µ—Ä–µ–º–µ–Ω–Ω—É—é, —Ç–æ–≥–¥–∞ –±–µ–∑ —Ä–∞–∑–Ω–∏—Ü—ã, –∫–∞–∫—É—é —Ñ–æ—Ä–º—É –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:

–≠—Ç–∏ –∑–Ω–∞—á–µ–Ω–∏—è —Ä–∞–≤–Ω—ã –¥—Ä—É–≥ –¥—Ä—É–≥—É –∏ –Ω–µ —Ä–∞–≤–Ω—ã –Ω–∏–∫–∞–∫–∏–º –¥—Ä—É–≥–∏–º –∑–Ω–∞—á–µ–Ω–∏—è–º. –≠—Ç–æ —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–µ –ø—Ä–∞–≤–∏–ª–æ —è–∑—ã–∫–∞.

```js

```

–ü—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –∏ –¥—Ä—É–≥–∏—Ö –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è `<` `>` `<=` `>=`
–ó–Ω–∞—á–µ–Ω–∏—è `null/undefined` –ø—Ä–µ–æ–±—Ä–∞–∑—É—é—Ç—Å—è –∫ —á–∏—Å–ª–∞–º: `null` —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è `0`, –∞ `undefined` ‚Äì `NaN`.

### 6. **–°—Ç—Ä–∞–Ω–Ω—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å—Ä–∞–≤–Ω–µ–Ω–∏—è null –∏ 0**

–ü—Ä–∏—á–∏–Ω–∞ –≤ —Ç–æ–º, —á—Ç–æ –Ω–µ—Å—Ç—Ä–æ–≥–æ–µ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ –∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏—è `>` `<` `>=` `<=` —Ä–∞–±–æ—Ç–∞—é—Ç –ø–æ-—Ä–∞–∑–Ω–æ–º—É. –°—Ä–∞–≤–Ω–µ–Ω–∏—è –ø—Ä–µ–æ–±—Ä–∞–∑—É—é—Ç `null` –≤ —á–∏—Å–ª–æ, —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—è –µ–≥–æ –∫–∞–∫ `0`. –ü–æ—ç—Ç–æ–º—É –≤—ã—Ä–∞–∂–µ–Ω–∏–µ (3) `null >= 0` –∏—Å—Ç–∏–Ω–Ω–æ, –∞ `null > 0` –ª–æ–∂–Ω–æ.

–° –¥—Ä—É–≥–æ–π —Å—Ç–æ—Ä–æ–Ω—ã, –¥–ª—è –Ω–µ—Å—Ç—Ä–æ–≥–æ–≥–æ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ `==` –∑–Ω–∞—á–µ–Ω–∏–π `undefined` –∏ `null` –¥–µ–π—Å—Ç–≤—É–µ—Ç –æ—Å–æ–±–æ–µ –ø—Ä–∞–≤–∏–ª–æ: —ç—Ç–∏ –∑–Ω–∞—á–µ–Ω–∏—è –Ω–∏ –∫ —á–µ–º—É –Ω–µ –ø—Ä–∏–≤–æ–¥—è—Ç—Å—è, –æ–Ω–∏ —Ä–∞–≤–Ω—ã –¥—Ä—É–≥ –¥—Ä—É–≥—É –∏ –Ω–µ —Ä–∞–≤–Ω—ã –Ω–∏—á–µ–º—É –¥—Ä—É–≥–æ–º—É. –ü–æ—ç—Ç–æ–º—É (2) `null == 0` –ª–æ–∂–Ω–æ.

### 7. **–ù–µ—Å—Ä–∞–≤–Ω–µ–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ undefined**
