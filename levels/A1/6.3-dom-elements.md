# **6.3. Работа с элементами DOM**

## Поиск элементов в DOM

### Методы поиска элементов

#### `getElementById()`

Поиск элемента по уникальному идентификатору:

```javascript
// HTML: <div id="main-content">...</div>
const mainContent = document.getElementById("main-content");
console.log(mainContent); // элемент или null

// Сокращенная запись (не рекомендуется)
console.log(window["main-content"]); // тот же элемент
```

#### `getElementsByClassName()`

Поиск элементов по классу (возвращает HTMLCollection):

```javascript
// HTML: <div class="card">...</div><div class="card">...</div>
const cards = document.getElementsByClassName("card");
console.log(cards.length); // количество элементов
console.log(cards[0]); // первый элемент

// Поиск по нескольким классам
const activeCards = document.getElementsByClassName("card active");
```

#### `getElementsByTagName()`

Поиск элементов по имени тега:

```javascript
// Все параграфы
const paragraphs = document.getElementsByTagName("p");

// Все элементы
const allElements = document.getElementsByTagName("*");

// Поиск внутри конкретного элемента
const nav = document.getElementById("navigation");
const navLinks = nav.getElementsByTagName("a");
```

#### `querySelector()`

Поиск первого элемента по CSS-селектору:

```javascript
// По ID
const header = document.querySelector("#header");

// По классу
const firstCard = document.querySelector(".card");

// По тегу
const firstParagraph = document.querySelector("p");

// Сложные селекторы
const navLink = document.querySelector("nav ul li a");
const activeButton = document.querySelector("button.active");
const dataAttribute = document.querySelector('[data-role="button"]');

// Псевдоклассы
const firstChild = document.querySelector("li:first-child");
const lastChild = document.querySelector("li:last-child");
const nthChild = document.querySelector("li:nth-child(3)");
```

#### `querySelectorAll()`

Поиск всех элементов по CSS-селектору (возвращает NodeList):

```javascript
// Все элементы с классом
const allCards = document.querySelectorAll(".card");

// Комбинированные селекторы
const buttons = document.querySelectorAll('button, input[type="button"]');

// Сложные селекторы
const menuItems = document.querySelectorAll("nav > ul > li");

// Итерация по результатам
allCards.forEach((card, index) => {
  console.log(`Карточка ${index}:`, card);
});
```

### Современные методы поиска

#### `closest()`

Поиск ближайшего родительского элемента по селектору:

```javascript
const button = document.querySelector(".submit-btn");

// Найти ближайшую форму
const form = button.closest("form");

// Найти ближайший элемент с классом
const card = button.closest(".card");

// Если не найден, возвращает null
const modal = button.closest(".modal"); // может быть null
```

#### `matches()`

Проверка соответствия элемента селектору:

```javascript
const element = document.querySelector(".item");

if (element.matches(".active")) {
  console.log("Элемент активен");
}

if (element.matches('[data-type="important"]')) {
  console.log("Важный элемент");
}
```

## Изменение содержимого элементов

### `innerHTML`

Получение и установка HTML-содержимого:

```javascript
const container = document.getElementById("container");

// Получение содержимого
console.log(container.innerHTML);

// Установка нового содержимого
container.innerHTML = "<p>Новый параграф</p><span>Спан</span>";

// Добавление к существующему содержимому
container.innerHTML += "<div>Дополнительный блок</div>";

// Очистка содержимого
container.innerHTML = "";
```

**⚠️ Предупреждение:** `innerHTML` может быть небезопасен при работе с пользовательскими данными (XSS-атаки).

### `textContent`

Получение и установка текстового содержимого:

```javascript
const paragraph = document.querySelector("p");

// Получение текста (без HTML-тегов)
console.log(paragraph.textContent);

// Установка текста (HTML-теги экранируются)
paragraph.textContent = "Новый текст с <strong>тегами</strong>";

// Результат: "Новый текст с <strong>тегами</strong>" (теги видны как текст)
```

### `innerText`

Получение и установка видимого текста:

```javascript
const element = document.querySelector(".text");

// Различия между textContent и innerText
console.log(element.textContent); // Весь текст, включая скрытый
console.log(element.innerText); // Только видимый текст

// Пример с скрытым элементом
const hiddenSpan = document.querySelector(".hidden"); // display: none
console.log(hiddenSpan.textContent); // текст есть
console.log(hiddenSpan.innerText); // пустая строка
```

### `outerHTML`

Получение и замена элемента целиком:

```javascript
const element = document.querySelector(".replace-me");

// Получение элемента с его тегами
console.log(element.outerHTML);

// Замена всего элемента
element.outerHTML = '<div class="new-element">Новый элемент</div>';

// После этого переменная element больше не ссылается на элемент в DOM!
```

## Работа с атрибутами

### Стандартные атрибуты как свойства

```javascript
const link = document.querySelector("a");
const input = document.querySelector("input");

// Чтение атрибутов
console.log(link.href); // URL ссылки
console.log(link.target); // _blank, _self и т.д.
console.log(input.value); // значение поля
console.log(input.type); // тип поля
console.log(input.disabled); // true/false

// Изменение атрибутов
link.href = "https://example.com";
link.target = "_blank";
input.value = "Новое значение";
input.disabled = true;
```

### Методы для работы с атрибутами

#### `getAttribute()` и `setAttribute()`

```javascript
const element = document.querySelector(".my-element");

// Получение атрибута
const id = element.getAttribute("id");
const className = element.getAttribute("class");
const customAttr = element.getAttribute("data-custom");

// Установка атрибута
element.setAttribute("id", "new-id");
element.setAttribute("class", "new-class");
element.setAttribute("data-value", "123");

// Несуществующий атрибут возвращает null
const nonExistent = element.getAttribute("non-existent"); // null
```

#### `hasAttribute()` и `removeAttribute()`

```javascript
const element = document.querySelector("input");

// Проверка наличия атрибута
if (element.hasAttribute("required")) {
  console.log("Поле обязательно для заполнения");
}

// Удаление атрибута
element.removeAttribute("disabled");
element.removeAttribute("required");

// Условное добавление/удаление
if (someCondition) {
  element.setAttribute("required", "");
} else {
  element.removeAttribute("required");
}
```

#### `attributes`

Получение всех атрибутов элемента:

```javascript
const element = document.querySelector("img");

// Коллекция всех атрибутов
console.log(element.attributes);

// Перебор атрибутов
for (let attr of element.attributes) {
  console.log(`${attr.name}: ${attr.value}`);
}

// Или через Array.from
Array.from(element.attributes).forEach((attr) => {
  console.log(`${attr.name}: ${attr.value}`);
});
```

### Data-атрибуты

Специальные атрибуты для хранения пользовательских данных:

```javascript
// HTML: <div data-user-id="123" data-role="admin" data-last-login="2023-09-18">

const element = document.querySelector("[data-user-id]");

// Доступ через dataset (рекомендуемый способ)
console.log(element.dataset.userId); // "123"
console.log(element.dataset.role); // "admin"
console.log(element.dataset.lastLogin); // "2023-09-18"

// Изменение через dataset
element.dataset.userId = "456";
element.dataset.status = "active";

// Удаление data-атрибута
delete element.dataset.role;

// Доступ через getAttribute (альтернативный способ)
console.log(element.getAttribute("data-user-id"));
element.setAttribute("data-theme", "dark");
```

## Работа с классами

### Свойство `className`

```javascript
const element = document.querySelector(".my-element");

// Получение всех классов как строки
console.log(element.className); // "class1 class2 class3"

// Установка классов
element.className = "new-class another-class";

// Добавление класса к существующим
element.className += " additional-class";

// Очистка всех классов
element.className = "";
```

### Объект `classList`

Более удобный способ работы с классами:

```javascript
const element = document.querySelector(".my-element");

// Добавление класса
element.classList.add("new-class");
element.classList.add("class1", "class2", "class3"); // несколько сразу

// Удаление класса
element.classList.remove("old-class");
element.classList.remove("class1", "class2"); // несколько сразу

// Переключение класса
element.classList.toggle("active"); // добавит, если нет; удалит, если есть

// Переключение с условием
element.classList.toggle("visible", isVisible); // добавит только если isVisible = true

// Проверка наличия класса
if (element.classList.contains("active")) {
  console.log("Элемент активен");
}

// Замена класса
element.classList.replace("old-class", "new-class");

// Количество классов
console.log(element.classList.length);

// Итерация по классам
element.classList.forEach((className) => {
  console.log(className);
});
```

## Создание и добавление элементов

### `createElement()`

Создание новых элементов:

```javascript
// Создание элемента
const newDiv = document.createElement("div");

// Установка атрибутов и содержимого
newDiv.id = "new-element";
newDiv.className = "card active";
newDiv.textContent = "Новый элемент";

// Альтернативно через setAttribute
newDiv.setAttribute("data-type", "card");

// Создание более сложной структуры
const article = document.createElement("article");
const header = document.createElement("header");
const title = document.createElement("h2");
const content = document.createElement("p");

title.textContent = "Заголовок статьи";
content.textContent = "Содержимое статьи...";

header.appendChild(title);
article.appendChild(header);
article.appendChild(content);
```

### Добавление элементов в DOM

#### `appendChild()`

Добавление элемента как последнего дочернего:

```javascript
const container = document.getElementById("container");
const newElement = document.createElement("p");
newElement.textContent = "Новый параграф";

// Добавление в конец контейнера
container.appendChild(newElement);

// Перемещение существующего элемента
const existingElement = document.querySelector(".move-me");
container.appendChild(existingElement); // переместится в container
```

#### `insertBefore()`

Вставка элемента перед указанным:

```javascript
const container = document.getElementById("container");
const newElement = document.createElement("div");
const referenceElement = container.querySelector(".reference");

// Вставка перед reference элементом
container.insertBefore(newElement, referenceElement);

// Вставка в начало контейнера
container.insertBefore(newElement, container.firstChild);
```

#### Современные методы вставки

```javascript
const element = document.querySelector(".target");
const newElement = document.createElement("div");

// Вставка в разные позиции
element.insertAdjacentElement("beforebegin", newElement); // перед элементом
element.insertAdjacentElement("afterbegin", newElement); // в начало элемента
element.insertAdjacentElement("beforeend", newElement); // в конец элемента
element.insertAdjacentElement("afterend", newElement); // после элемента

// Вставка HTML-строки
element.insertAdjacentHTML("beforeend", "<p>Новый параграф</p>");

// Вставка текста
element.insertAdjacentText("afterbegin", "Текст в начале");
```

### `cloneNode()`

Клонирование элементов:

```javascript
const original = document.querySelector(".template");

// Поверхностное клонирование (только сам элемент)
const shallowClone = original.cloneNode(false);

// Глубокое клонирование (с дочерними элементами)
const deepClone = original.cloneNode(true);

// Добавление клона в DOM
document.body.appendChild(deepClone);
```

## Удаление элементов

### `removeChild()`

Удаление дочернего элемента:

```javascript
const parent = document.getElementById("parent");
const child = document.getElementById("child");

// Удаление дочернего элемента
parent.removeChild(child);

// Удаление с проверкой существования
if (parent.contains(child)) {
  parent.removeChild(child);
}
```

### `remove()`

Современный способ удаления элемента:

```javascript
const element = document.querySelector(".delete-me");

// Простое удаление
element.remove();

// С проверкой существования
if (element) {
  element.remove();
}
```

### `replaceChild()`

Замена одного элемента другим:

```javascript
const parent = document.getElementById("parent");
const oldChild = document.getElementById("old");
const newChild = document.createElement("div");
newChild.textContent = "Новый элемент";

// Замена элемента
parent.replaceChild(newChild, oldChild);
```

## Практические примеры

### Динамический список задач

```javascript
class TodoList {
  constructor(containerId) {
    this.container = document.getElementById(containerId);
    this.tasks = [];
    this.taskCounter = 0;
  }

  addTask(text) {
    const task = {
      id: ++this.taskCounter,
      text: text,
      completed: false,
    };

    this.tasks.push(task);
    this.renderTask(task);
  }

  renderTask(task) {
    // Создание структуры задачи
    const taskElement = document.createElement("div");
    taskElement.className = "task-item";
    taskElement.dataset.taskId = task.id;

    const checkbox = document.createElement("input");
    checkbox.type = "checkbox";
    checkbox.checked = task.completed;
    checkbox.addEventListener("change", () => this.toggleTask(task.id));

    const text = document.createElement("span");
    text.textContent = task.text;
    text.className = "task-text";

    const deleteBtn = document.createElement("button");
    deleteBtn.textContent = "×";
    deleteBtn.className = "delete-btn";
    deleteBtn.addEventListener("click", () => this.deleteTask(task.id));

    // Сборка элемента
    taskElement.appendChild(checkbox);
    taskElement.appendChild(text);
    taskElement.appendChild(deleteBtn);

    // Добавление в контейнер
    this.container.appendChild(taskElement);
  }

  toggleTask(id) {
    const task = this.tasks.find((t) => t.id === id);
    if (task) {
      task.completed = !task.completed;
      this.updateTaskElement(id);
    }
  }

  deleteTask(id) {
    // Удаление из массива
    this.tasks = this.tasks.filter((t) => t.id !== id);

    // Удаление из DOM
    const taskElement = this.container.querySelector(`[data-task-id="${id}"]`);
    if (taskElement) {
      taskElement.remove();
    }
  }

  updateTaskElement(id) {
    const task = this.tasks.find((t) => t.id === id);
    const taskElement = this.container.querySelector(`[data-task-id="${id}"]`);

    if (task && taskElement) {
      taskElement.classList.toggle("completed", task.completed);
    }
  }

  clear() {
    this.tasks = [];
    this.container.innerHTML = "";
  }
}

// Использование
const todoList = new TodoList("todo-container");
todoList.addTask("Изучить DOM");
todoList.addTask("Создать проект");
```

### Фильтрация и поиск элементов

```javascript
class ElementFilter {
  constructor(containerId, inputId) {
    this.container = document.getElementById(containerId);
    this.searchInput = document.getElementById(inputId);
    this.originalElements = Array.from(this.container.children);

    this.setupEventListeners();
  }

  setupEventListeners() {
    this.searchInput.addEventListener("input", (e) => {
      this.filter(e.target.value);
    });
  }

  filter(query) {
    const lowerQuery = query.toLowerCase();

    this.originalElements.forEach((element) => {
      const text = element.textContent.toLowerCase();
      const matches = text.includes(lowerQuery);

      if (matches) {
        element.style.display = "";
        this.highlightMatches(element, query);
      } else {
        element.style.display = "none";
      }
    });
  }

  highlightMatches(element, query) {
    if (!query) {
      // Убираем подсветку если запрос пустой
      element.innerHTML = element.textContent;
      return;
    }

    const text = element.textContent;
    const regex = new RegExp(`(${query})`, "gi");
    const highlightedText = text.replace(regex, "<mark>$1</mark>");

    element.innerHTML = highlightedText;
  }

  reset() {
    this.searchInput.value = "";
    this.filter("");
  }
}

// Использование
const filter = new ElementFilter("items-container", "search-input");
```

### Accordion компонент

```javascript
class Accordion {
  constructor(containerSelector) {
    this.container = document.querySelector(containerSelector);
    this.items = [];
    this.setupEventListeners();
  }

  addItem(title, content) {
    const item = this.createAccordionItem(title, content);
    this.container.appendChild(item);
    this.items.push(item);
  }

  createAccordionItem(title, content) {
    const item = document.createElement("div");
    item.className = "accordion-item";

    const header = document.createElement("div");
    header.className = "accordion-header";
    header.textContent = title;
    header.setAttribute("role", "button");
    header.setAttribute("aria-expanded", "false");

    const body = document.createElement("div");
    body.className = "accordion-body";
    body.innerHTML = content;
    body.style.display = "none";

    item.appendChild(header);
    item.appendChild(body);

    return item;
  }

  setupEventListeners() {
    this.container.addEventListener("click", (e) => {
      if (e.target.classList.contains("accordion-header")) {
        this.toggleItem(e.target);
      }
    });
  }

  toggleItem(header) {
    const body = header.nextElementSibling;
    const isOpen = body.style.display !== "none";

    // Закрываем все остальные элементы
    this.closeAll();

    if (!isOpen) {
      body.style.display = "block";
      header.classList.add("active");
      header.setAttribute("aria-expanded", "true");
    }
  }

  closeAll() {
    this.items.forEach((item) => {
      const header = item.querySelector(".accordion-header");
      const body = item.querySelector(".accordion-body");

      body.style.display = "none";
      header.classList.remove("active");
      header.setAttribute("aria-expanded", "false");
    });
  }
}

// Использование
const accordion = new Accordion(".accordion");
accordion.addItem("Секция 1", "<p>Содержимое первой секции</p>");
accordion.addItem("Секция 2", "<p>Содержимое второй секции</p>");
```
