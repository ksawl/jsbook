### Атрибут `target` в элементах `<form>`

Атрибут `target` также используется в элементах `<form>` для указания того, как будет обрабатываться отправка формы.

1. **`_self`**: Отправляет форму в том же окне или вкладке. Это значение по умолчанию.
    ```html
    <form action="/submit" method="post" target="_self">
        <!-- Поля формы -->
        <input type="submit" value="Отправить" />
    </form>
    ```
2. **`_blank`**: Отправляет форму в новом окне или новой вкладке браузера.
    ```html
    <form action="/submit" method="post" target="_blank">
        <!-- Поля формы -->
        <input type="submit" value="Отправить" />
    </form>
    ```
3. **`_parent`**: Отправляет форму в родительском окне или фрейме.
    ```html
    <form action="/submit" method="post" target="_parent">
        <!-- Поля формы -->
        <input type="submit" value="Отправить" />
    </form>
    ```
4. **`_top`**: Отправляет форму во всем текущем окне браузера, отменяя все фреймы.
    ```html
    <form action="/submit" method="post" target="_top">
        <!-- Поля формы -->
        <input type="submit" value="Отправить" />
    </form>
    ```
5. **Имя фрейма**: Отправляет форму в указанном фрейме.
    ```html
    <form action="/submit" method="post" target="contentFrame">
        <!-- Поля формы -->
        <input type="submit" value="Отправить" />
    </form>
    ```

#### Пример использования:

```html
<!DOCTYPE html>
<html lang="ru">
    <head>
        <meta charset="UTF-8" />
        <title>Пример использования target в формах</title>
    </head>
    <body>
        <h1>Пример использования атрибута target в формах</h1>
        <p>Форма отправится в том же окне:</p>
        <form action="/submit" method="post" target="_self">
            <input type="text" name="name" placeholder="Введите ваше имя" />
            <input type="submit" value="Отправить" />
        </form>

        <p>Форма отправится в новой вкладке:</p>
        <form action="/submit" method="post" target="_blank">
            <input type="text" name="name" placeholder="Введите ваше имя" />
            <input type="submit" value="Отправить" />
        </form>

        <p>Форма отправится в родительском окне (если есть фрейм):</p>
        <form action="/submit" method="post" target="_parent">
            <input type="text" name="name" placeholder="Введите ваше имя" />
            <input type="submit" value="Отправить" />
        </form>

        <p>Форма отправится во всем окне:</p>
        <form action="/submit" method="post" target="_top">
            <input type="text" name="name" placeholder="Введите ваше имя" />
            <input type="submit" value="Отправить" />
        </form>

        <p>Форма отправится в указанном фрейме:</p>
        <iframe name="contentFrame" src="" width="600" height="400"></iframe>
        <form action="/submit" method="post" target="contentFrame">
            <input type="text" name="name" placeholder="Введите ваше имя" />
            <input type="submit" value="Отправить" />
        </form>
    </body>
</html>
```
